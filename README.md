# House_Party

A music app that lets you create or join listening parties with your friends! 

![banner_image](https://i.imgur.com/S8I1QNd.png)

## Technologies Used
* HTML/JS/CSS
* Bootstrap
* ReactJS
* Material-UI
* Python
* Django
* SQL
* Spotify API
